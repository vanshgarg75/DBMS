SQL SELF JOIN

A self join is a regular join, but the table is joined with itself.

Syntax

SELECT column_names
FROM table1 T1, table1 T2
WHERE condition;

Demo Database

In this example, we will use the "Employees" table:

+----+-------+-----------+----------+
| Id | Name | Position | ManagerId |
+----+-------+-----------+----------+
| 1 | John | CEO | null |
| 2 | Jane | CTO | 1 |
| 3 | Dave | Developer | 2 |
| 4 | Sara | Developer | 2 |
| 5 | Mike | Manager | 1 |
+----+-------+-----------+----------+

SQL SELF JOIN Example

The following SQL statement matches employees with their managers:

SELECT E.Name AS Employee, M.Name AS Manager
FROM Employees E, Employees M
WHERE E.ManagerId = M.Id;

Result

+----------+---------+
| Employee | Manager |
+----------+---------+
| Jane | John |
| Dave | Jane |
| Sara | Jane |
| Mike | John |
+----------+---------+
