Date and Time Functions

MySQL provides a wide variety of date and time functions to work with temporal data types. These functions help you retrieve, format, calculate, and manipulate date and time values efficiently.

Common Date and Time Functions

Function Description Example
CURDATE() Returns the current date.
SELECT CURDATE();
CURTIME() Returns the current time.
SELECT CURTIME();
NOW() Returns the current date and time.
SELECT NOW();
DATE(expr) Extracts the date part of a datetime expression.
SELECT DATE('2025-04-08 10:30:00');
TIME(expr) Extracts the time part of a datetime expression.
SELECT TIME('2025-04-08 10:30:00');
YEAR(date) Returns the year of a date.
SELECT YEAR('2025-04-08');
MONTH(date) Returns the month of a date.
SELECT MONTH('2025-04-08');
DAY(date) Returns the day of the month.
SELECT DAY('2025-04-08');
HOUR(time) Returns the hour of a time.
SELECT HOUR('10:30:45');
MINUTE(time) Returns the minute of a time.
SELECT MINUTE('10:30:45');
SECOND(time) Returns the seconds of a time.
SELECT SECOND('10:30:45');
DATEDIFF(expr1, expr2) Returns the number of days between two dates.
SELECT DATEDIFF('2025-04-08', '2025-04-01');
DATE_ADD(date, INTERVAL expr unit) Adds a time interval to a date.
SELECT DATE_ADD('2025-04-08', INTERVAL 10 DAY);
DATE_SUB(date, INTERVAL expr unit) Subtracts a time interval from a date.
SELECT DATE_SUB('2025-04-08', INTERVAL 5 DAY);
STR_TO_DATE(str, format) Converts a string to a date/time value based on a given format.
SELECT STR_TO_DATE('08-04-2025', '%d-%m-%Y');
DATE_FORMAT(date, format) Formats a date according to a given format.
SELECT DATE_FORMAT('2025-04-08', '%W %M %Y');

Conclusion

These functions are vital for handling, formatting, and calculating with temporal data. MySQL 8.4 offers powerful tools to make date and time manipulation simple and effective for developers and analysts.
