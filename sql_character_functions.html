SQL String Functions

SQL provides a variety of string functions to perform operations on string data types. These functions allow you to manipulate and retrieve information from character strings effectively.

Common SQL String Functions

Function Description Example
ASCII() Returns the ASCII value of the first character in a string.
SELECT ASCII('A') AS AsciiValue;
CHAR() Returns the character corresponding to the ASCII code.
SELECT CHAR(65) AS Character;
CHARINDEX() Returns the starting position of a substring within a string.
SELECT CHARINDEX('world', 'Hello world') AS Position;
CONCAT() Concatenates two or more strings into one.
SELECT CONCAT('Hello', ' ', 'World') AS Greeting;
LEFT() Returns the left part of a string with the specified number of characters.
SELECT LEFT('Hello World', 5) AS LeftPart;
RIGHT() Returns the right part of a string with the specified number of characters.
SELECT RIGHT('Hello World', 5) AS RightPart;
LEN() Returns the number of characters in a string.
SELECT LEN('Hello World') AS Length;
LOWER() Converts all characters in a string to lowercase.
SELECT LOWER('HELLO WORLD') AS Lowercase;
UPPER() Converts all characters in a string to uppercase.
SELECT UPPER('hello world') AS Uppercase;
LTRIM() Removes leading spaces from a string.
SELECT LTRIM(' Hello World') AS Trimmed;
RTRIM() Removes trailing spaces from a string.
SELECT RTRIM('Hello World ') AS Trimmed;
REPLACE() Replaces all occurrences of a substring within a string with another substring.
SELECT REPLACE('Hello World', 'World', 'SQL') AS Replaced;
REVERSE() Reverses the characters in a string.
SELECT REVERSE('Hello') AS Reversed;
STUFF() Deletes a specified length of characters in a string and inserts another set of characters at a specified starting point.
SELECT STUFF('Hello World', 6, 5, 'SQL') AS Stuffed;
SUBSTRING() Extracts a portion of a string starting at a specified position and of a specified length.
SELECT SUBSTRING('Hello World', 7, 5) AS Extracted;

Conclusion

SQL string functions are powerful tools that enable you to work with and manipulate text data. Mastering these functions is essential for effective querying, data cleaning, and formatting.
