PL/SQL Control Structures

Control structures in PL/SQL are used to control the flow of execution within a program. There are three main types of control structures:

  • Conditional Control (IF, CASE)
  • Iterative Control (LOOP, WHILE, FOR)
  • Sequential Control (GOTO, NULL)

1. Conditional Control

IF-THEN

IF condition THEN -- statements END IF;

IF-THEN-ELSE

IF condition THEN -- statements ELSE -- alternative statements END IF;

IF-THEN-ELSIF

IF condition1 THEN -- statements ELSIF condition2 THEN -- statements ELSE -- default statements END IF;

Example

DECLARE grade CHAR(1) := 'B'; BEGIN IF grade = 'A' THEN DBMS_OUTPUT.PUT_LINE('Excellent'); ELSIF grade = 'B' THEN DBMS_OUTPUT.PUT_LINE('Very Good'); ELSE DBMS_OUTPUT.PUT_LINE('Needs Improvement'); END IF; END;

CASE Statement

CASE selector WHEN 'A' THEN statement1; WHEN 'B' THEN statement2; ELSE default_statement; END CASE;
DECLARE grade CHAR(1) := 'A'; BEGIN CASE grade WHEN 'A' THEN DBMS_OUTPUT.PUT_LINE('Excellent'); WHEN 'B' THEN DBMS_OUTPUT.PUT_LINE('Very Good'); ELSE DBMS_OUTPUT.PUT_LINE('Other'); END CASE; END;

2. Iterative Control

Basic LOOP

LOOP -- statements EXIT WHEN condition; END LOOP;
DECLARE i NUMBER := 1; BEGIN LOOP DBMS_OUTPUT.PUT_LINE('i = ' || i); i := i + 1; EXIT WHEN i > 5; END LOOP; END;

WHILE LOOP

WHILE condition LOOP -- statements END LOOP;
DECLARE i NUMBER := 1; BEGIN WHILE i <= 5 LOOP DBMS_OUTPUT.PUT_LINE('i = ' || i); i := i + 1; END LOOP; END;

FOR LOOP

FOR counter IN [REVERSE] lower_bound..upper_bound LOOP -- statements END LOOP;
BEGIN FOR i IN 1..5 LOOP DBMS_OUTPUT.PUT_LINE('i = ' || i); END LOOP; END;

3. Sequential Control

GOTO Statement

GOTO label; ... <
DECLARE x NUMBER := 10; BEGIN IF x = 10 THEN GOTO skip; END IF; DBMS_OUTPUT.PUT_LINE('This will not be printed'); <> DBMS_OUTPUT.PUT_LINE('Skipped to here'); END;

NULL Statement

Used as a placeholder when no action is needed.

IF salary IS NULL THEN NULL; -- do nothing ELSE DBMS_OUTPUT.PUT_LINE('Salary is not null'); END IF;

Conclusion

Control structures allow developers to write logic-driven programs that can make decisions and repeat actions. PL/SQL offers a full set of features for writing efficient procedural code.
