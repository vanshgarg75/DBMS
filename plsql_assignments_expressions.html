PL/SQL Assignment Statement

An assignment statement in PL/SQL sets the value of a variable, a field, or an element of a collection. The value of the expression on the right-hand side is assigned to the target on the left-hand side using the assignment operator :=.

Syntax

target := expression;

target can be a variable, a field in a record, or an element in a collection.

expression can be any legal PL/SQL expression that returns a value compatible with the type of the target.

Example

DECLARE bonus NUMBER(5,2); salary NUMBER(7,2) := 4500.00; BEGIN bonus := salary * 0.10; DBMS_OUTPUT.PUT_LINE('Bonus: ' || bonus); END;

Explanation

  • A variable salary is initialized to 4500.00.
  • bonus is calculated by multiplying salary by 0.10 using the assignment statement bonus := salary * 0.10;.
  • The value of bonus is then displayed using DBMS_OUTPUT.PUT_LINE.

Notes

  • The target of the assignment must be writable. Constants and expressions cannot be used as the target.
  • The data type of the expression must be compatible with the data type of the target variable.
  • Assignment can occur between same-type variables, or implicit conversion may be applied if types are compatible.
